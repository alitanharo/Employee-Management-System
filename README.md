# Employee Management System

This is a simple Employee Management System web application built using React and Redux.

## Features

- Create new employees with details like first name, surname, email, birth date, job title, and status.
- Validate employee data using Formik and Yup.
- Utilize Redux for state management.
- Styled with styled-components for a consistent and attractive UI.
- React DatePicker for selecting birth dates.
- Tooltips for enhanced interactivity.

## Getting Started

### Prerequisites

- Node.js and npm should be installed on your machine.

### Installation

1. Clone the repository:


  git clone https://github.com/alitanharo/Employee-Management-System.git

  Navigate to the project directory:


  cd employee-management-system

  Install dependencies:


  npm install

  Start the development server:


  npm start
